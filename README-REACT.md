## Getting Started

### Install Dependencies

```bash
npm install
```

### Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Why React + Vite?

- **10x faster** than traditional build tools
- **Hot Module Replacement (HMR)** for instant updates
- **Component-based** architecture for maintainability
- **Modern JavaScript** with ES modules
- **Optimized builds** with tree-shaking and code splitting

## Migration from Vanilla JS

This project was converted from vanilla HTML/CSS/JS to React + Vite for:
- Better performance with Virtual DOM
- Easier state management
- Reusable components
- Modern development experience
- Production-ready optimizations
